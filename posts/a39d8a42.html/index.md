# 文本编辑器问题汇总

## Notepad++怎么设置tab替换为4个空格

`设置` -> `首选项...` -> `语言菜单` -> `制表符设置` -> 勾选上`转换为空格`
<!--more-->

## UltraEdit - 怎么显示文件标签栏和侧边栏

### 显示文件标签栏

`view` -> `views/lists` -> `open Files Tabs`

### 显示侧边栏

`view` -> `views/lists` -> `File Tree View`
